Detailed Description

A condition variable. It is used to synchronize a Task with either one or multiple Task and/or interrupt service routines.

Warning
Only a Task is allowed to wait, wait_for or wait_until. An interrupt service routine is NOT allowed to call wait in any case
Remarks
It is a replacement for std::condition_variable
