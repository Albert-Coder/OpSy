Detailed Description

template<class Item, class Interface = Item>
class opsy::EmbeddedList< Item, Interface >

A linked list container that uses the contained item to store forward and backward pointers.

Template Parameters
ItemThe contained item type, which must inherit EmbeddedNode of itself
InterfaceIn case of multiple inheritance, the interface to use to access forward and backward pointer. By default, the Item type itself
Remarks
It is not copy constructible not copy assignable, but it is move constructible and move assignable
