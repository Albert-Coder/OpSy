Detailed Description

template<std::size_t StorageSize, typename ReturnType, typename ... Arguments>
class opsy::Callback< ReturnType(Arguments...), StorageSize >

A callback. A callback is used as a way to defer code execution. For example, when you ask a peripheral to do something, you may want the peripheral to execute code when it has finished, calling you back (hence the name) so that you can e.g. set a flag.

Template Parameters
StorageSizeThe size available to store the callback data (in void* increment)
ReturnTypeThe type of the return value (if none, use void)
ArgumentsThe arguments of the callback (id none, use void)
Remarks
It is a replacement for std::function
